### 3.3 API-003 GET /trips/generate/status（P1）

#### 功能验收

| 用例ID       | 用例描述                 | 预期结果                                                     |
| ------------ | ------------------------ | ------------------------------------------------------------ |
| API-003-F-01 | 查询 pending 任务状态    | 返回 `{ status: 'pending', position: 1 }`                    |
| API-003-F-02 | 查询 processing 任务状态 | 返回 `{ status: 'processing', currentDay: 2, totalDays: 5 }` |
| API-003-F-03 | 查询 completed 任务状态  | 返回 `{ status: 'completed', tripId, completedAt }`          |
| API-003-F-04 | 查询 failed 任务状态     | 返回 `{ status: 'failed', errorMessage }`                    |

#### 边界验收

| 用例ID       | 用例描述                       | 预期结果                                           |
| ------------ | ------------------------------ | -------------------------------------------------- |
| API-003-B-01 | taskId 不存在                  | 返回 404 `{ error: 'TaskNotFound' }`               |
| API-003-B-02 | 非本人 taskId                  | 返回 403 `{ error: 'Forbidden' }`                  |
| API-003-B-03 | 任务创建超过 30 分钟仍 pending | 返回 `{ status: 'failed', reason: 'TaskTimeout' }` |

#### 单元/集成测试

```typescript
// API-003 单元测试
it('应返回正确的进度百分比', async () => {
  const task = await createProcessingTask({ currentDay: 2, totalDays: 5 });
  const res = await request(app).get(`/api/trips/generate/status?taskId=${task.id}`);
  expect(res.body.progress).toBe(40); // 2/5 * 100
});
```

### 3.7 API-007 PUT /trips/{tripId}/day/{dayIndex}（P1）

#### 功能验收

| 用例ID       | 用例描述            | 预期结果                                                      |
| ------------ | ------------------- | ------------------------------------------------------------- |
| API-007-F-01 | 更新时间段成功      | 返回 200，DB 中对应 Item 时间已更新                           |
| API-007-F-02 | 删除一个 Item       | Body 含 `deletedItemIds: ['xxx']`，返回 200，该 Item 不再出现 |
| API-007-F-03 | 调整顺序（reorder） | Body 含 `itemOrder: ['id3','id1','id2']`，返回 200，顺序更新  |

#### 边界验收

| 用例ID       | 用例描述                       | 预期结果                                                       |
| ------------ | ------------------------------ | -------------------------------------------------------------- | ---------------------------------------------- |
| API-007-B-01 | 时间重叠                       | 返回 400 `{ error: 'TimeOverlap', overlappingItemIds: [...] }` |
| API-007-B-02 | 修改后的总活动时间超过可用时间 | 返回 400 `{ error: 'Overtime' }`，但前端可强制保存             |
| API-007-B-03 | 并发编辑同一天                 | 两个请求同时到达                                               | 后到的请求返回 409 `{ error: 'EditConflict' }` |

#### 单元/集成测试

```typescript
it('应拒绝时间重叠的编辑', async () => {
  const res = await request(app)
    .put(`/api/trips/${tripId}/day/0`)
    .send({
      timelineItems: [
        { startTime: '09:00', endTime: '12:00' },
        { startTime: '10:00', endTime: '11:00' },
      ],
    });
  expect(res.status).toBe(400);
  expect(res.body.error).toBe('TimeOverlap');
});
```
