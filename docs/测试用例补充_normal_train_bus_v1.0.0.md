## 6. API 接口测试

### 6.1 POST /transport/search 接口

**请求示例 - 普速火车**

```json
POST /api/v1/transport/search

Body:
{
  "fromCity": "北京",
  "toCity": "长沙",
  "date": "2026-07-01",
  "prefer": ["normal_train"],     // 偏好普速火车
  "departTimePeriod": "evening",  // 偏好夜间出发（卧铺车次）
  "passengers": {
    "adults": 2,
    "children": 0,
    "elders": 0
  }
}
```

**响应示例 - 普速火车**

```json
{
  "code": 0,
  "data": {
    "options": [
      {
        "type": "normal_train",
        "trainNumber": "Z1",
        "departTime": "18:20",
        "arriveTime": "次日 08:15",
        "durationMinutes": 835,
        "departureStation": "北京西站",
        "arrivalStation": "长沙站",
        "seatTypes": ["硬座", "硬卧", "软卧"],
        "seatPrices": { "硬座": 156, "硬卧": 280, "软卧": 450 },
        "availableSeats": {
          "硬座": 120,
          "硬卧": 30,
          "软卧": 10
        },
        "isOvernight": true,
        "note": "隔夜车次，含卧铺"

}
```
