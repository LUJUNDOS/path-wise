## 三、初始化脚本

### 3.1 一键初始化脚本（Windows PowerShell）

**文件路径**：`scripts/init-project.ps1`

```powershell
# scripts/init-project.ps1
# PATH-WISE 项目初始化脚本（Windows PowerShell）
# 执行方式：右键 -> 使用 PowerShell 运行（管理员权限）

Write-Host "🚀 PATH-WISE 项目初始化开始..." -ForegroundColor Cyan

# 1. 创建项目根目录
$ProjectName = "path-wise"
if (-not (Test-Path $ProjectName)) {
    New-Item -ItemType Directory -Path $ProjectName | Out-Null
    Write-Host "✅ 创建项目目录: $ProjectName" -ForegroundColor Green
} else {
    Write-Host "⚠️ 项目目录已存在: $ProjectName" -ForegroundColor Yellow
}

Set-Location $ProjectName

# 2. 初始化 Git 仓库
if (-not (Test-Path ".git")) {
    git init
    Write-Host "✅ 初始化 Git 仓库" -ForegroundColor Green
}

# 3. 创建 Monorepo 结构
Write-Host "📁 创建 Monorepo 目录结构..." -ForegroundColor Cyan

# 创建目录
$directories = @(
    "apps/web/src/components",
    "apps/web/src/pages",
    "apps/web/src/hooks",
    "apps/web/src/lib",
    "apps/web/src/types",
    "apps/web/public",
    "apps/api/src/routes",
    "apps/api/src/plugins",
    "apps/api/src/services",
    "apps/api/src/adapters",
    "apps/api/src/types",
    "apps/api/src/utils",
    "apps/api/prisma",
    "packages/shared/src/types",
    "packages/shared/src/utils",
    "docker/postgres",
    "docker/redis",
    "scripts",
    "docs"
)

foreach ($dir in $directories) {
    if (-not (Test-Path $dir)) {
        New-Item -ItemType Directory -Path $dir -Force | Out-Null
    }
}

Write-Host "✅ 目录结构创建完成" -ForegroundColor Green

# 4. 创建 pnpm-workspace.yaml
```

## 五、Prisma Schema 设计

### 5.1 数据库 Schema（基于《数据库设计规格书\_v1.0.0.md》）

**文件路径**：`apps/api/prisma/schema.prisma`

```prisma
// Prisma Schema for PATH-WISE
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

// 用户表
model User {
  id          Int      @id @default(autoincrement())
  openid      String?  @unique // 微信 OpenID（阶段二）
  phone       String?  @unique
  nickname    String?
  avatarUrl   String?
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt

  trips Trip[]

  @@map("users")
}

// 行程表
model Trip {
  id            Int      @id @default(autoincrement())
  userId        Int?
  title         String   // 行程标题
  cityFrom      String   // 出发城市
  cityTo        String[] // 目的地城市（支持多目的地）
  startDate     DateTime // 出发日期
  endDate       DateTime // 返回日期（null 表示单程）
  totalDays     Int      // 总天数
  transportTo   String   // 大交通偏好（high_speed_rail/normal_train/flight/bus/auto）
  transportLocal String   // 当地交通偏好（public/rental/walking/taxi）
  budget        Int?     // 总预算（元）
  travelers     Json     // 旅客信息（{adults, children, elders}）
  status        String   @default("generating") // generating/generated/failed
  progress      Int      @default(0) // 生成进度（0-100）
  failReason    String?  // 失败原因
  dayPlans      Json?    // 每日行程（JSONB）
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt

  user User? @relation(fields: [userId], references: [id])

  @@map("trips")
}

// 城市知识库表（静态数据）
model CityKnowledge {
  id              Int      @id @default(autoincrement())
  cityName        String   @unique
  province        String?
  level           String?  // 一线城市/新一线/二线/三线
  tags            String[] // 标签数组
  intercityTransport Json? // 大交通数据（JSONB）
  localTransport  Json?    // 当地交通数据（JSONB）
  accommodation   Json?    // 住宿数据（JSONB）
  poiList         Json?    // POI 列表（JSONB）
  weatherPattern  Json?    // 天气规律（JSONB）
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt

  @@map("city_knowledge")
}
```

## 七、常见问题（FAQ）

### Q1：Windows 开发环境遇到权限问题？

**A**：以管理员身份运行 PowerShell，或启用 WSL2。

### Q2：pnpm 安装失败？

**A**：先安装 pnpm：

```bash
npm install -g pnpm
```

### Q3：Docker Desktop 启动失败？

**A**：确保 BIOS 启用虚拟化，或切换到 Linux 容器模式。

### Q4：Prisma Migrate 失败？

**A**：检查 `.env` 文件中的 `DATABASE_URL` 是否正确，确保 PostgreSQL 容器运行中。
